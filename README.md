# L'Italie sur roues 🇮🇹

A static restaurant website for **L'Italie sur roues** — a French-Italian restaurant with a great sense of humour and even better food.

## Pages

- **Accueil** (`index.html`) — Hero landing + full menu (Entrées, Plats principaux, Desserts)
- **À Propos** (`about.html`) — The story, philosophy, and team behind the restaurant

## Structure

```
├── index.html          # Home page with menu
├── about.html          # About page
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── main.js         # Navbar scroll, mobile toggle, scroll animations
├── images/
│   └── team.jpg        # Team photo (add your own)
├── LICENSE             # MIT License
└── README.md
```

## Setup

Just open `index.html` in a browser — no build tools required.

To add the team photo, place your image at `images/team.jpg`.
